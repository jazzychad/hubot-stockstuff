#hubot-stockstuff

A hubot script for stock quotes and charts

## Installation

Add **hubot-stockstuff** to your `package.json` file:

```json
"dependencies": {
  "hubot": ">= 2.5.1",
  "hubot-scripts": ">= 2.4.2",
  "hubot-stockstuff": ">= 0.0.0"
}
```

Add **hubot-stockstuff** to your `external-scripts.json`:

```json
["hubot-stockstuff"]
```

Run `npm install hubot-stockstuff`
